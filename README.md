# Многошаговая форма на Flask + Jinja2

Простой проект веб-формы с несколькими шагами, построенный на Flask и Jinja2.

## Описание

Этот проект представляет собой многошаговую форму (wizard form) с 4 шагами:
1. **Шаг 1**: Личная информация (имя, email, телефон)
2. **Шаг 2**: Дополнительная информация (возраст, город, пол)
3. **Шаг 3**: Выбор опций (интересы, рассылка, комментарии)
4. **Шаг 4**: Подтверждение и итог

Данные сохраняются в сессии Flask между шагами, что позволяет пользователю возвращаться назад и изменять введенную информацию. После подтверждения формы все данные сохраняются в JSON файл `data/submissions.json`.

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите приложение

```bash
python app.py
```

### 3. Откройте в браузере

Перейдите по адресу: http://127.0.0.1:5000

## Особенности

- ✅ Многошаговая форма с навигацией вперед/назад
- ✅ Сохранение данных между шагами через Flask sessions
- ✅ **Автоматическое сохранение данных в JSON файл** после подтверждения
- ✅ Визуальный индикатор текущего шага
- ✅ Валидация данных на каждом шаге
- ✅ Адаптивный дизайн
- ✅ Красивый современный UI

## Сохранение данных

Все отправленные формы автоматически сохраняются в файл `data/submissions.json`. Каждая запись содержит:
- Временную метку отправки
- Все данные, введенные пользователем

Файл создается автоматически при первой отправке формы. Все данные хранятся в формате JSON с читаемым форматированием.

## Настройка
